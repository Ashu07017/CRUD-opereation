# 📌 CRUD Operations using Node.js, Express & MongoDB

## 📖 Project Description

This project demonstrates a full-stack CRUD (Create, Read, Update, Delete) application using:

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)

The application allows users to manage student records.

---

## 🚀 Features

* ➕ Add new student
* 📄 View all students
* ✏️ Update student details
* 🗑 Delete student

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```
git clone https://github.com/Ashu07017/CRUD-opereation.git
cd CRUD-opereation
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
npx nodemon server.js
```

---

### 3️⃣ Run Frontend

Open:

```
frontend/index.html
```

in your browser.

## 🔗 API Endpoints

| Method | Endpoint          | Description        |
| ------ | ----------------- | ------------------ |
| POST   | /api/students     | Create new student |
| GET    | /api/students     | Get all students   |
| GET    | /api/students/:id | Get single student |
| PUT    | /api/students/:id | Update student     |
| DELETE | /api/students/:id | Delete student     |
